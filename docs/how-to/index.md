# How-to guides

How-to guides are problem-oriented content that shows you how to accomplish a
specific goal. They assume you already have a working egide instance.

## Available guides

- [Production deployment](../guides/production.md): harden egide for production use.
- [High availability](../guides/high-availability.md): run egide with multiple replicas.
- [Backup and recovery](../guides/backup.md): back up the vault and restore from a snapshot.
- [Docker Compose](../guides/docker.md): manage egide with Docker Compose.

More guides are planned. See [ROADMAP.md](../../ROADMAP.md) for upcoming topics.
